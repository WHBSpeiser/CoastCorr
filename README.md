## Installation

### Create Environment

```bash
conda env create --file env/requirements.yml

conda activate coast_corr 
```

### Authenticate Earth Engine/Drive

```bash
earthengine authenticate 
```

### Run Jupyter

```bash
jupyter notebook 
```
